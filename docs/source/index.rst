Bienvenido a Bit Alert!
===================================

**Bit Alert** es un sistema informático desarrollado para optimizar recursos y procesos difíciles en el comercio de criptomonedas, diseñado para una fácil comprensión y manejo de las múltiples herramientas que Bit Alert puede ofrecer: 

•	Transacciones manuales
•	Transacciones automáticas (basadas en un previo análisis)
•	Gestión y manejo de las billeteras electrónicas
•	Sección de Bit Alertnalizadas que describe momentos claves y oportunidades para poder comercializar una criptomoneda en específico llegando en tiempo real hasta tu dispositivo móvil. 

Bit Alert este escrito de una forma robusta en Python, CSV y es un sistema de objeto orientado, está diseñado para trabajar de manera portable, porque utiliza un sistema de información en CSV, Para controlar la información. Siendo a si capaz de funcionar en plataformas de:

•	Windows 
•	Mac
•	Linux (Todas sus versiones posibles)

Bit Alert busca ser parte de día a día laboral en las criptomonedas y con un óptimo uso del sistema se pueden lograr obtener muy buenos resultados economicos.  Este sistema contribuye a todos los nuevos usuarios en los comercios de criptomonedas, es la herramienta que se necesita para poder entrar a este mundo sin tener previa experiencia o sin ser un profesional en el tema. 



Lee :doc:`usage` la seccion para mas informacion, que incluye su forma de instalacion :ref:`installation` the project.

.. note::

   Este sistema esta bajo desarrollo constante y activo.
